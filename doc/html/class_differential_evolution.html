Detailed Description

template<class Tinput, class Toutput>
class DifferentialEvolution< Tinput, Toutput >

Template Differential Evolution class.

Differential Evolution (DE) is based on the strategy that employs the difference of tow randomly selected parameter vectors as the source of random variations for a third parameter vector.
